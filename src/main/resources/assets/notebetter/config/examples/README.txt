NoteBetter Examples
===================

This is a list of NoteBetter JSON examples. All examples listed here come with a short description.

vanilla
-------

The NoteBetter JSON which best describes vanilla note block behaviour.

soniex2
-------

The NoteBetter JSON by SoniEx2. Provides a handful additional instruments, such as:

- "block.note.pling", triggered by an iron block. This is an unused vanilla instrument.
- The experience orb touch sound, triggered by an enchanting table.
- The anvil landing sound, triggered by an anvil.
- The piston sound, triggered by a piston. To get the note block to play without triggering the piston, put the piston facing up, towards the note block.
- The egg sound, triggered by a brewing stand.
